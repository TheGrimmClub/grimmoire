# GRIMMoire · Python-Lernpfad

Willkommen im Zauberbuch des **Grimm Clubs** — ein Python-Kurs für Anfänger, der
dich vom ersten Befehl in der Shell bis zu deinem eigenen veröffentlichten Paket
führt.

## Fang hier an

Vor dem Code die Kultur. Lies das **[Manifest der Lernkultur](00-manifest.md)** —
das Gelöbnis der Gilde über Vertrauen, die gemeinsame Werkbank und das Fragen
ohne Scham. Es ist das Fundament, auf dem alles andere steht.

## Richte deine Werkbank ein

Vor dem ersten Kapitel drei kurze Einrichtungsschritte — die machst du nur
einmal:

1. **[uv installieren](a__01-setup-uv.md)** — das eine Werkzeug für alles
2. **[Python installieren](a__02-setup-python.md)** — die Sprache selbst
3. **[Installation testen](a__03-test-installation.md)** — beweisen, dass es läuft

> Schon an Tag eins [GrimmSetup](https://github.com/TheGrimmClub/GrimmSetup)
> durchgearbeitet? Dann ist das erledigt — fang bei Kapitel 01 an.

## Der Pfad

Die Reise verläuft in elf Schritten, jeder ein kurzes Kapitel:

1. **Befehle & Aufrufe** — mit der Maschine sprechen
2. **Imports** — nutzen, was andere gebaut haben
3. **argparse** — Eingaben von der Kommandozeile lesen
4. **Variablen & Werte** — sich Dinge merken
5. **Debugger** — sehen, was dein Code wirklich tut
6. **Bedingungen** — Entscheidungen treffen
7. **Schleifen & Steuerung** — Dinge wiederholen
8. **Funktionen** — einem Stück Arbeit einen Namen geben
9. **Klassen** — deine eigenen Baupläne (triff `grimm.Actor`)
10. **Werkzeug · uv** — Python auf moderne Art verwalten
11. **Baue dein Paket** — etwas Echtes veröffentlichen

> Neue Kapitel kommen dazu, während das Camp voranschreitet. Nutze die
> Navigation links, um zu springen.
